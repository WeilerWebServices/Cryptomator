Cryptomator 1.3.0 RC 3

When we first started with Cryptomator, we had all the source code in one big project. In preparation for the FUSE/Dokany integration, we wanted to modularize its core components. In fact, we had to completely rewrite Cryptomator. The cryptographic code is now available as CryptoLib and CryptoFS. With that, we're able to write more manageable adapters like WebDAV and FUSE/Dokany for version 1.4.

This is a prerelease and is only intended for testing. Wait for the stable release for production purposes.

Full release notes are available here.

Download Prerelease Build

We'd like to keep the distribution simple. We're offering binaries for Windows (64/32 Bit), macOS, and Debian (64 Bit).

  • Download Cryptomator 1.3.0-rc3 for Windows (64 Bit)
  • Download Cryptomator 1.3.0-rc3 for Windows (32 Bit)
  • Download Cryptomator 1.3.0-rc3 for macOS
  • Download Cryptomator 1.3.0-rc3 for Debian (64 Bit)

Please use a test vault or ideally a backup/copy of your productive vault. If Cryptomator isn't behaving as expected and the issue isn't reproducible with version 1.2, please let us know.

Other Systems

If your system is not listed, you can also download version 1.3.0-rc3 as a JAR file here, which requires the following installed on your system:

  • Java 8 (min. 8u51, we recommend to use the current version)
  • JCE unlimited strength policy files (needed for 256-bit keys)
