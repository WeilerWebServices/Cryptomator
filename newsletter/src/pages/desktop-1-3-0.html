Cryptomator 1.3

We finally implemented the most-wished feature: Auto-Unlock! Still experimental and will be completed by autostart (optionally hidden) in future versions.

We've completely rewritten Cryptomator to achieve major under-the-hood improvements. Its core components are now modularized to CryptoLib, CryptoFS, and WebDAV-NIO-Adapter.

Full release notes and more details are available in our blog post.

Cryptomator for iOS & Android

Cryptomator 1.2.5 for iOS and Cryptomator 0.6.0 for Android have been released as well.

If you've updated to Cryptomator 1.2.4 for iOS before, we're very sorry for the "missing folders" bug. Please update to 1.2.5 and the issue should be resolved.

We're currently preparing Cryptomator 0.6.1 for Android which will improve local storage support for Android 5.0 or higher. Stay tuned if you're experiencing issues with the new local storage mechanism.
